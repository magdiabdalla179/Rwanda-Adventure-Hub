## Project Description

Rwanda Adventure Hub is a no-code e-commerce website designed for selling outdoor and adventure equipment online. The website was developed using Wix, a no-code website development platform that allows users to create websites using drag-and-drop tools without writing code.

The store focuses on providing reliable and affordable adventure products such as camping tents, hiking backpacks, sleeping bags, camping lanterns, and water bottles. The website allows customers to browse products, view prices, read product descriptions, contact the business, and interact with a shopping cart.

## Objective of the Project

The main objective of this project is to design and develop a simple e-commerce website using a no-code platform. The project demonstrates the basic features of an online store, including product display, navigation, contact information, and cart functionality.

## Platform Used

The website was developed using **Wix**.

Wix was selected because it is beginner-friendly and provides ready-made tools for creating websites, adding products, customizing pages, and enabling e-commerce features such as product listings and cart interaction.

## Website Pages

The Rwanda Adventure Hub website includes the following pages:

### 1. Home Page

The Home page introduces the business with the store name **Rwanda Adventure Hub** and a welcome message. It also includes a professional adventure-themed design, featured products, navigation menu, and call-to-action buttons that encourage users to explore the store.

### 2. Product Page

The Product page displays the available outdoor and adventure products. Each product includes an image, product name, price, short description, and an Add to Cart button.

### 3. About Page

The About page explains the background of Rwanda Adventure Hub, its mission, business goals, and commitment to customers. It presents the store as a reliable place for customers looking for outdoor and adventure equipment.

### 4. Contact Page

The Contact page provides business contact information and allows customers to communicate with the store. It includes phone number, email address, location, and a contact form.

### 5. Cart Page

The Cart page allows users to view products they have added to the cart. It provides basic e-commerce interaction such as reviewing selected products, adjusting quantities, and removing items from the cart.

## Challenges Faced

During the development of this project, several challenges were encountered. One challenge was selecting a suitable website template that matched the adventure and outdoor equipment theme. Another challenge was finding suitable product images and organizing the products clearly on the website.

It was also necessary to make sure that the website layout was responsive and that the shopping cart interaction worked properly. Creating clear product descriptions and arranging the website pages professionally also required careful attention.
